# Модуль «Права доступа»

![Permissions Icon](static/Permissions.png)

Полная документация: [docs/README.ru.md](docs/README.ru.md).

Плагин **Permissions** управляет доступом к HTTP-маршрутам: ограничения из кода, дополнительные правила для пользователей и ролей, сброс правил.

| | |
|---|---|
| **Админка** | [/admin/Permissions](/admin/Permissions) |
| **Категория** | System |
| **Версия** | 0.1 |

## Быстрый старт

1. Откройте **Админка → Права доступа** (`/admin/Permissions`).
2. Разверните карточку модуля — для каждого HTTP-метода виден **Доступ по умолчанию**.
3. **Изменить** или **Настроить модуль** — дополнительные правила разрешения/запрета.
4. **Сбросить правила** (↺) — удалить доп. правила, останется только доступ по умолчанию.
